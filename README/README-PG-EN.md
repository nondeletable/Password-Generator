## ğŸ” What this app does

- Generate strong passwords **offline** (no telemetry, no tracking).
- Use ready-to-go **policies** (Standard / Admin / NIST).
- Avoid copy/paste risks with **auto-hide** and **clipboard auto-clean**.
- Get a password that matches your needs (length, symbols, digits, etc.).
- Keep it simple: portable ".exe", minimal UI, quick workflow.

## ğŸ¨ Features

- Cryptographically secure generation via Python `secrets` (not `random`).
- Custom character sets: letters / digits / symbols.
- Guaranteed inclusion: at least one char from each selected category.
- Password strength indicator (weak / fair / strong).
- **Auto-hide** shown password after **10 seconds**.
- **Clipboard protection**: clears clipboard after **30 seconds**.
- One-click copy.
- Built-in **Password Policies**:
  - **Standard** - balanced complexity
  - **Admin** - stricter rules for privileged accounts
  - **NIST** - inspired by NIST SP 800-63B (longer, user-friendly, secure)
